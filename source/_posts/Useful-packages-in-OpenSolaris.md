To install these packages, open a Terminal and type:

```sh
pfexec pkg install SUNWlexpt sunstudio SUNWgmake \
	gcc-dev-4 SUNWflexlex SUNWgnu-gettext SUNWlynx \
	SUNWgnome-img-editor SUNWGtk SUNWhexedit SUNWgftp \
	SUNWrdesktop SUNWsmba SUNWsmbs SUNWsmbskr \
	SUNWtree SUNWtsclient SUNWtvnc xvm-gui \
	SUNWwxwidgets SUNWcdrw SUNWPython26
```

`openoffice` is optional

Uninstall SUNWlldap

```sh
pfexec pkg uninstall SUNWlldap
```
